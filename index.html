Sustentación

Este proyecto web está diseñado con una estructura sencilla y clara, donde cada sección presenta contenido multimedia relacionado con el trabajo realizado, como animaciones, un álbum de fotos y un folleto. La navegación es fácil, con enlaces que permiten al usuario explorar diferentes partes del proyecto, incluyendo videos incrustados desde YouTube. Se ha utilizado HTML para la estructura y se asume que el CSS se emplea para la presentación visual, garantizando una experiencia interactiva y visualmente atractiva.

Las herramientas utilizadas incluyen plataformas como Animaker para las animaciones, GIMP e Inkscape para la creación de imágenes y gráficos, lo que refleja un enfoque integral y creativo en el proceso de desarrollo del proyecto. La página busca ofrecer una experiencia completa al visitante, proporcionando tanto una visión detallada del trabajo como una representación visual del mismo a través de diversos elementos multimedia.
